Timed Text Standards

What’s the Timed Text?

Timed text is the presentation of text media in synchrony with other media, such as audio and video. Src

W3C standards for Timed text

  • WebVTT = Web Video Timed Text
  • TTML = Timed Text Markup Language

WebVTT Format

File extension is vtt. This format defines a list of “cues” that have a start time, an end time, and text to display.
